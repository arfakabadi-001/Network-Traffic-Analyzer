# NetPulse- Network Traffic Analyzer

A real-time network traffic monitoring and analysis tool built using Python.  
This project captures live network packets, analyzes traffic behavior, detects anomalies, and presents structured summaries with visual insights.

---

## Overview

The Network Traffic Analyzer simulates a real-world monitoring system by filtering unnecessary packet noise, grouping traffic intelligently, and generating meaningful alerts instead of excessive or unrealistic outputs.  
It is designed to be resume-ready, examiner-friendly, and scalable.

---

## Key Features

- Real-time packet capture and inspection  
- Source and destination IP analysis  
- Port-wise traffic monitoring  
- Threshold-based alert detection  
- Time-based traffic summarization  
- Clean, readable terminal output  
- Persistent data storage using SQLite  
- Traffic visualization using Matplotlib  

---

## Tech Stack

- Python  
- Scapy – packet capturing and inspection  
- SQLite – persistent storage  
- Matplotlib – traffic visualization  
- Threading – real-time processing
- Flask – Python backend
- HTML5, CSS3 – User Interface  
---

## How It Works

1. Captures live network packets  
2. Filters unnecessary traffic to avoid noise  
3. Groups packets into time-based summaries  
4. Detects abnormal traffic patterns  
5. Logs alerts and stores summaries in a database  
6. Generates visual traffic reports
